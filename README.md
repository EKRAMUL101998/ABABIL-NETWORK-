# ABABIL Network

ABABIL Network is a blockchain built with the Cosmos SDK.

## Network Information

- Network Name: ABABIL Network
- Chain ID: ababil-1
- Binary: ababild
- Denom: uababil

## Requirements

- Go 1.26+
- Git

## Build

```bash
make install
```

## Run

```bash
ababild start
```
